# fpm - Foundry Package Manager

An unofficial CLI package manager for [FoundryVTT](https://foundryvtt.com), written in Rust.

Foundry's package management becomes painful quickly if you've ever tried moving your installation between machines
or had to install/update modules on a slow or unstable network.

> **Early development.** The commands that exist work, but most of the planned features are still being built. Don't use this without making backups!
